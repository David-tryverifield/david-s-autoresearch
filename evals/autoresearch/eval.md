# Rubric — autoresearch

Q1: Did autoresearch follow all 7 steps in order without skipping? (Y/N)
  - A ceiling run (Step 4 says "Ceiling achieved") still earns Y if all 7 step labels are present and Step 7 says "No rewrite — skill at ceiling."

Q2: Were scoring subagents isolated (no cross-run or skill-file access referenced)? (Y/N)

Q3: Was the set-summary diagnosis specific and tied to a concrete pattern in the scores? (1–5)
  - Ceiling case: "Ceiling achieved — all criteria at max" earns a 3 if the scores confirm it, 4–5 if the run also notes which reps were consistent.

Q4: Was the rewrite decision correct and targeted? (1–5)
  - If a weak point was diagnosed: score how well the rewrite addressed only that point (1–5).
  - If ceiling was correctly diagnosed: a clear "No rewrite — skill at ceiling" earns a 4. Skipping Step 7 entirely earns a 1.

Q5: Was the final report table accurate and complete for the sets run so far? (Y/N)
  - A single-row baseline table is complete and accurate for a first-set run — score Y.
  - Score N only if scores are wrong, sets are missing from prior history, or "pending" rows appear.
